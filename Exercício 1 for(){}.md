# Exercício 1 for(){}

**1.** Repita 10x a frase olá mundo.

```java
public class classefor {
    public static void main(String[] args) {
        for (int i = 0; i < 10; i++) { // Repete 10 vezes
            System.out.println("Olá mundo!");
        }
    }
}
```
