# Exercício 2 for (){}

**2.** Crie um programa que conte de 15 a 50.

```java
public class Contador {
    public static void main(String[] args) {
        for (int i = 15; i <= 50; i++) { // Começa em 15 e vai até 50
            System.out.println(i);
        }
    }
}
```
